# アドオン (Add-ons)

MythicMobsはAPIを通じてMythicの機能を拡張するアドオンを持っています。

## 公式アドオン

### [Crucible](https://mythiccraft.io/index.php?resources/crucible-create-unbelievable-mythic-items.2/)

Crucibleを使用すると、Mythicのスキルシステムを利用して信じられないようなカスタムアイテムを作成できます：

- カスタムアイテムの作成（カスタムブロック、カスタム家具を含む）
- アイテムスキルとトリガー
- カスタムアーマーセット
- カスタムモデルとテクスチャのサポート

### [MythicEnchants](https://mythiccraft.io/index.php?resources/mythic-enchants.6/)

Mythicのスキルシステムを使ったカスタムエンチャントシステム：

- カスタムエンチャントを作成
- エンチャントにスキルを割り当て
- プレイヤーベースのスキルシステム

### [MythicDungeons](https://mythiccraft.io/index.php?resources/mythicdungeons.12/)

カスタムインスタンス化されたダンジョンシステム：

- 複数グループの同時プレイが可能
- カスタムMythicMobsとの完全な統合
- ダンジョンのリセット機能
- 報酬と進行システム

### [MythicAchievements](https://mythiccraft.io/index.php?resources/mythicachievements.21/)

カスタム実績システム：

- MythicMobsと完全な統合
- カスタム報酬

### [ModelEngine](https://www.spigotmc.org/resources/conxeptworks-model-engine%E2%80%94ultimate-entity-model-manager-1-16-5-1-21-4.79477/)

エンティティにカスタム3Dモデルを追加する公式サポートアドオン：

- カスタム3Dモデルの表示
- アニメーションのサポート
- MythicMobsとの統合

### [MCPets](https://www.spigotmc.org/resources/mcpets.97590/)

カスタムペットシステム：

- MythicMobsと統合されたペット

## コミュニティアドオン

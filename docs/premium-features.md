# プレミアム機能 (Premium Features)

## プレミアム (Premium)

[MythicMobs Premium](http://www.mythicmobs.net/index.php?account/upgrades)には多くのプレミアム限定の特典が含まれています：

- Discordでの特別なPremiumロール
- プレミアムサポートチャンネルへのアクセス
- 最新バージョンへの早期アクセス
- 開発ビルドへのアクセス
- [レイトレース](skills/mechanics/raytrace.md)や[チェーンミサイル](skills/mechanics/chainmissile.md)などのプレミアム限定メカニクスへのアクセス！
- カスタムダメージタイプとダメージ修飾子の使用能力
- 以下での数学と[プレースホルダー](skills/placeholders.md)の使用能力：
  - モブ属性（HP、ダメージなど）
  - ほとんどのスキル
  - アイテム属性
  - ドロップテーブルの数量
- [スキルパラメーター](skills/meta-skills.md)の使用能力
- [インラインターゲット条件](skills/inline-conditions.md)の使用能力
- [プロジェクタイルスキル](skills/mechanics/projectile.md)でのヒット条件とバウンスの使用能力
- 条件付き[AIゴール/ターゲッター](mobs/custom-ai.md)の使用能力
- 任意のメカニクスで`origin=@targeter`の使用能力
- さらに多くの機能！

MythicMobs Premiumでロック解除された新しい機能はすべて、Crucible、RPG、Enchants、Achievements、Dungeonsなどの互換性のあるMythicプラグインでも使用できます！

プレミアム限定機能は今後も追加し続けられます。[こちらをクリック](http://www.mythicmobs.net/index.php?account/upgrades)してアカウントをPremiumにアップグレードできます。

## プレミアム+ (Premium+)

大規模なサーバーを持つ方やさらに多く貢献したい方向けに、Premium+パッケージで追加の特典があります：

- Discordでの特別なPremium+ロール
- 開発者をpingできるサポートチケットの優先対応
- 所有する複数のサーバーで使用可能なライセンス

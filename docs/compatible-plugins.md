# 互換プラグイン (Compatible Plugins)

MythicMobsはMinecraftの多くの人気プラグインと互換性があり、APIを通じて機能をさらに拡張するアドオンもあります。

## Mythicアドオン

- **[Crucible](https://mythiccraft.io/index.php?resources/crucible-create-unbelievable-mythic-items.2/)** - Mythicのスキルシステムを使って信じられないようなカスタムアイテムを作成し、モブに対してプレイヤーに必要な強みを与えます！

- **[BossShop Add-on](https://www.spigotmc.org/resources/mythicmobs-%E2%9A%94-bossshoppro-add-on-%E2%9A%94.58415/)** - 人気のショッププラグインBossShopProとの互換性を提供し、ショップでMythicMobsのアイテムを簡単に使用できるようにします。

- **[Depenizen](https://ci.citizensnpcs.co/view/All/job/Depenizen/)** - DepenizenmはMythicMobsのタグと機能を[Denizen](https://guide.denizenscript.com/)に追加し、DenizenのコンディションとターゲッターをMythicMobsに追加します！

- **[Quests Add-on](https://www.spigotmc.org/resources/mythicmobs-quests-module.7261/)** - 人気のQuestsプラグインとの互換性を提供し、Mythic Mobsを目標としてクエストを達成できるようにします。

- **[Skript Add-on](https://github.com/BerndiVader/mmSkriptAddon)** - Skriptとの完全なクロスオーバー互換性を追加します！

# @ForwardWall (前方壁ターゲッター)

## 概要
キャスターの前方に平面（壁）を形成し、そのブロック群の位置をターゲットにします。

- **タイプ**: 複数場所ターゲッター

## 属性
| 属性 | 省略形 | 説明 | デフォルト |
|------|--------|------|-----------|
| forward | f, amount, a | 前方への距離（小数可） | 5.0 |
| yoffset | y | 平面のY軸オフセット（小数可） | 0.0 |
| height | h | 平面の高さ（整数） | 2 |
| width | w | 平面の幅（整数） | 3 |

## 使用例
```yaml
ExampleSkill:
  Skills:
  - effect:particles @ForwardWall{f=5;y=1;h=2;w=2}  # 前方5ブロック、2x2の壁にパーティクル
```

## 備考
- 前方に壁状のエフェクトや障壁を作るのに適しています
- `setblock` メカニクスと組み合わせて実際にブロックを設置することもできます

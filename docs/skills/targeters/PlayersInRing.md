# @PlayersInRing (リング内プレイヤーターゲッター)

## 概要
キャスターを中心としたリング（ドーナツ形）の範囲内にいるすべてのプレイヤーをターゲットにします。

- **タイプ**: 複数エンティティターゲッター
- **省略形**: `@pring`

## 属性
| 属性 | 省略形 | 説明 | デフォルト |
|------|--------|------|-----------|
| minrange | min | リングの最小半径（ブロック数） | 5 |
| maxrange | max | リングの最大半径（ブロック数） | 10 |

## 使用例
```yaml
ExampleSkill:
  Skills:
  - ignite @PlayersInRing{min=2;max=10}  # キャスターから2〜10ブロックのリング内のプレイヤーに火をつける
```

## 備考
- 特定の距離範囲内のプレイヤーだけをターゲットにする場合に便利です
- `@EntitiesInRing` のプレイヤー限定版です

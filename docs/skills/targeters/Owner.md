# @Owner (オーナーターゲッター)

## 概要
キャスターモブのオーナーをターゲットにします。オーナーは `SetOwner` メカニクスで設定できます。

- **タイプ**: 単一エンティティターゲッター

## 属性
> *このターゲッターには属性がありません*

## 使用例
```yaml
ExampleSkill:
  Skills:
  - message{m="Hello there!"} @Owner  # オーナーにメッセージを送信
```

## 備考
- オーナーは `SetOwner` メカニクスを使って設定します
- バニラのオーナーシップ（オオカミなど）とは異なります

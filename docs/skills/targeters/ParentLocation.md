# @ParentLocation (親エンティティ位置ターゲッター)

## 概要
キャスターモブの親エンティティの位置をターゲットにします。

- **タイプ**: 単一場所ターゲッター
- **省略形**: `@summonerlocation`

## 属性
> *このターゲッターには属性がありません*

## 使用例
```yaml
ExampleSkill:
  Skills:
  - teleport @ParentLocation  # 親エンティティの位置にテレポート
```

## 備考
- `@Parent` はエンティティをターゲットにしますが、こちらは位置をターゲットにします
- 召喚モブが召喚者の元に戻るスキルパターンに使えます

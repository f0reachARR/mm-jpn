# @WolfOwner (オオカミオーナーターゲッター)

## 概要
キャスターがオオカミの場合、そのオオカミのバニラオーナーをターゲットにします。

- **タイプ**: 単一エンティティターゲッター

## 属性
> *このターゲッターには属性がありません*

## 使用例
```yaml
ExampleSkill:
  Skills:
  - heal{a=10} @WolfOwner  # オオカミのオーナーを回復
```

## 備考
- キャスターがバニラのオオカミである必要があります
- MythicMobsの `@Owner` ターゲッターとは異なります（バニラのオーナーシップを使用）

# @Parent (親ターゲッター)

## 概要
キャスターモブの親エンティティをターゲットにします。

- **タイプ**: 単一エンティティターゲッター
- **省略形**: `@summoner`

## 属性
> *このターゲッターには属性がありません*

## 使用例
```yaml
ExampleSkill:
  Skills:
  - command{c="say <target.name>"} @Parent  # 親エンティティの名前をチャットに表示
```

## 備考
- `summon` メカニクスで召喚されたモブの親をターゲットにします
- `@Children` ターゲッターの逆の関係です

# @Origin (原点ターゲッター)

## 概要
メタスキルの「原点」または「ソース」の位置をターゲットにします。通常はキャスターのモブですが、プロジェクタイルスキルなどでは異なる場合があります。

- **タイプ**: 単一場所ターゲッター
- **省略形**: `@source`

## 属性
> *このターゲッターには属性がありません*

## 使用例
```yaml
ExampleSkill:
  Skills:
  - effect:particles @origin  # 原点の位置にパーティクルを表示
```

## 備考
- 通常のスキル発動時: キャスターの位置
- プロジェクタイルスキルの `onTick` 内: プロジェクタイルの現在位置
- メカニクスやユニバーサル属性でスキルの原点が変更された場合: その変更後の位置
- `@SelfLocation` と異なり、コンテキストに応じて位置が変化します

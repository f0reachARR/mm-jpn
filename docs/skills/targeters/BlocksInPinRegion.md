# @BlocksInPinRegion (ピンリージョン内ブロックターゲッター)

## 概要
2つのピン（Pin）で区切られた領域内のすべてのブロックをターゲットにします。

- **タイプ**: 複数場所ターゲッター

## 属性
| 属性 | 省略形 | 説明 | デフォルト |
|------|--------|------|-----------|
| pin1 | p1 | 最初のピン | |
| pin2 | p2 | 2番目のピン | |

## 使用例
```yaml
ExampleSkill:
  Skills:
  - particle @BlocksInPinRegion{p1=mypin;p2=anotherpin}  # 2つのピンが定義する直方体内のブロックにパーティクル
```

## 備考
- ピン（Pin）機能については [Pins](/Pins) を参照してください
- 2つのピンが対角の頂点を定義する直方体（AABB）内のブロックが対象です
- 大規模なブロック操作に適しています

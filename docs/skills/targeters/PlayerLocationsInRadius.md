# @PlayerLocationsInRadius (半径内プレイヤー位置ターゲッター)

## 概要
キャスターを中心とした指定半径内にいるすべてのプレイヤーの位置をターゲットにします。

- **タイプ**: 複数場所ターゲッター
- **省略形**: `@LocationRadius`, `@PLIR`, `@PLR`

## 属性
| 属性 | 省略形 | 説明 | デフォルト |
|------|--------|------|-----------|
| radius | r | ターゲッターの半径（ブロック数） | 5 |
| yoffset | y | ターゲット位置のY軸オフセット | 0 |

## 使用例
```yaml
ExampleSkill:
  Skills:
  - effect:particles @PlayerLocationsInRadius{r=10}  # 半径10ブロック内のプレイヤー全員の位置にパーティクル
```

## 備考
- `@PlayersInRadius` はエンティティをターゲットにしますが、こちらは位置をターゲットにします
- 各プレイヤーの足元にエフェクトを出すのに使えます

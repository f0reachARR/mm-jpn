# @NearestPlayer (最近傍プレイヤーターゲッター)

## 概要
指定した半径内で最も近いプレイヤーをターゲットにします。

- **タイプ**: 単一エンティティターゲッター

## 属性
| 属性 | 省略形 | 説明 | デフォルト |
|------|--------|------|-----------|
| radius | r | ターゲッターの半径（ブロック数） | 5 |

## 使用例
```yaml
ExampleSkill:
  Skills:
  - settarget @NearestPlayer{r=10}  # 半径10ブロック内の最も近いプレイヤーをターゲットに設定
```

## 備考
- 半径内にプレイヤーが存在しない場合、スキルは発動しません

# @Target (ターゲットターゲッター)

## 概要
キャスターの現在のターゲットをターゲットにします。

- **タイプ**: 単一エンティティターゲッター
- **省略形**: `@T`

## 属性
> *このターゲッターには属性がありません*

## 使用例
```yaml
ExampleSkill:
  Skills:
  - ignite @Target         # キャスターのターゲットに火をつける
```

## 備考
- キャスターがモブの場合: そのモブのターゲットをターゲットにします
- キャスターがプレイヤーの場合: プレイヤーが見ているエンティティ（十分近い場合）をターゲットにします
- ガストのように、ハードコードされたAIのせいでターゲットを持てないエンティティには機能しません

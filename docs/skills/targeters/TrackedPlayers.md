# @TrackedPlayers (追跡プレイヤーターゲッター)

## 概要
キャスターのレンダリング距離内にいるプレイヤーをターゲットにします。

- **タイプ**: 複数エンティティターゲッター
- **省略形**: `@tracked`

## 属性
> *このターゲッターには属性がありません*

## 使用例
```yaml
ExampleSkill:
  Skills:
  - ignite @TrackedPlayers  # レンダリング距離内のすべてのプレイヤーに火をつける
```

## 備考
- プレイヤーのクライアントにレンダリングされているプレイヤーが対象です
- レンダリング距離はサーバーの設定に依存します

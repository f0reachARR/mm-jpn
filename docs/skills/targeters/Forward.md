# @Forward (前方位置ターゲッター)

## 概要
キャスターの前方にある指定距離の位置をターゲットにします。

- **タイプ**: 単一場所ターゲッター

## 属性
| 属性 | 省略形 | 説明 | デフォルト |
|------|--------|------|-----------|
| forward | f, amount, a | ターゲット地点までの距離（ブロック数） | 5 |
| rotate | rot | キャスターを中心としたターゲット位置の回転（度） | 0 |
| useeyelocation | uel | 目線位置をベースとして使用するか | false |
| lockpitch | | ピッチを0として扱うか（水平方向のみ） | false |

## 使用例
```yaml
ExampleSkill:
  Skills:
  - effect:particles @Forward{f=3;uel=true;rot=10}  # 前方3ブロック、目線基準、10度回転した位置にパーティクル
```

## 備考
- `lockpitch=true` にすると水平方向のみに前進します
- `rotate` で左右にオフセットを加えることができます

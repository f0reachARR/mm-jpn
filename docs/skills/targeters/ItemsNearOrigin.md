# @ItemsNearOrigin (原点付近アイテムターゲッター)

## 概要
メタスキルの原点付近にあるすべてのアイテム（ドロップされたアイテムエンティティ）をターゲットにします。

- **タイプ**: 複数エンティティターゲッター
- **省略形**: `@INO`

## 属性
| 属性 | 省略形 | 説明 | デフォルト |
|------|--------|------|-----------|
| radius | r | ターゲッターの半径（ブロック数） | 5 |

## 使用例
```yaml
ExampleSkill:
  Skills:
  - projectile{...;
    onTick=[
      - pickupitem @ItemsNearOrigin{r=5}
      # プロジェクタイルの進行中、原点から5ブロック以内のアイテムを拾う
    ]}
```

## 備考
- ドロップされたアイテムエンティティをターゲットにします
- `@ItemsInRadius` はキャスター基準、こちらはスキルの原点基準です

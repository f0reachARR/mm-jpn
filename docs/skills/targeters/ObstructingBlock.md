# @ObstructingBlock (障害ブロックターゲッター)

## 概要
キャスターの前方にある、移動を妨げているブロックをターゲットにします。

- **タイプ**: 単一場所ターゲッター

## 属性
> *このターゲッターには属性がありません*

## 使用例
```yaml
ExampleSkill:
  Skills:
  - effect:particles{hs=0.6;vs=0.6;a=20} @ObstructingBlock  # 障害ブロックにパーティクルを表示
```

## 備考
- キャスターが進めないブロックをターゲットにします
- ブロック破壊スキルと組み合わせると、前方のブロックを自動で破壊できます

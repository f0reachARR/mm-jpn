# @EscapeLocation (逃走位置ターゲッター)

## 概要
エンダーマンがテレポートを選択するような方法で、近くの安全な逃走位置をターゲットにします。

- **タイプ**: 単一場所ターゲッター
- **省略形**: `@EndermanTeleport`

## 属性
| 属性 | 省略形 | 説明 | デフォルト |
|------|--------|------|-----------|
| radius | r | テレポート先を探す水平・垂直の最大オフセット | 32 |
| iterations | i | テレポート先の探索試行回数 | 64 |
| height | h | テレポート先に必要な空気ブロックの高さ | 3 |

## 使用例
```yaml
ExampleSkill:
  Skills:
  - teleport @escapelocation{radius=32;iterations=64;height=3}  # 安全な逃走位置にテレポート
```

## 備考
- 危険な状況から逃げるスキルパターンに最適です
- エンダーマンのテレポートと同じロジックを使用します

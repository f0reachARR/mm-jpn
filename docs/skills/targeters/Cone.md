# @Cone (コーン形状位置ターゲッター)

## 概要
キャスターの前方にコーン（円錐）形状を形成し、その内部のランダムな位置をターゲットにします。Y軸には固定されており、上下方向には回転できません。

- **タイプ**: 複数場所ターゲッター

## 属性
| 属性 | 省略形 | 説明 | デフォルト |
|------|--------|------|-----------|
| angle | a | コーンの角度（度） | 90 |
| range | r | コーンの長さ（ブロック数） | 16 |
| points | p | コーン内に生成するポイント数 | angle×range×0.1 |
| slices | s | コーンをrange方向に分割するレイヤー数 | floor(range) |
| minpoints | mp | 各スライスの最小ポイント数 | 1 |
| rotation | rot | コーンの回転（度） | 0 |
| yoffset | yo, y | コーンのY軸オフセット | 0 |
| exact | e | 均一分布のための精密モードを使用するか | false |

## 使用例
```yaml
ExampleSkill:
  Skills:
  - effect:particles @Cone{a=45;r=10;p=200}  # 前方45度、10ブロックのコーンに200個のパーティクル
```

## 備考
- `@LivingInCone` はエンティティをターゲットにしますが、こちらは位置をターゲットにします
- `exact=true` で均一にポイントを分布させることができます

# @Trigger (トリガーターゲッター)

## 概要
スキルツリーをトリガーしたエンティティをターゲットにします。

- **タイプ**: 単一エンティティターゲッター

## 属性
> *このターゲッターには属性がありません*

## 使用例
```yaml
TomTheMurderous:
  Type: COW
  Skills:
  - damage{a=20} @trigger ~onInteract  # インタラクトしたエンティティにダメージを与える
```

## 備考
- `~onInteract` や `~onDamaged` などのトリガーと組み合わせて使うことが多いです
- スキルをトリガーしたエンティティ（攻撃したプレイヤー、インタラクトしたプレイヤーなど）に効果を適用します

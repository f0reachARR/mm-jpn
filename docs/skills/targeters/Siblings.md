# @Siblings (兄弟エンティティターゲッター)

## 概要
キャスターと同じ親を持つすべてのエンティティをターゲットにします。

- **タイプ**: 複数エンティティターゲッター
- **省略形**: `@sibling`, `@brothers`, `@sisters`

## 属性
> *このターゲッターには属性がありません*

## 使用例
```yaml
ExampleSkill:
  Skills:
  - ignite @Siblings  # 同じ親を持つすべてのエンティティに火をつける
```

## 備考
- 同じ親モブから召喚されたモブをグループとして操作する場合に便利です
- キャスター自身は含まれません（兄弟のみ）

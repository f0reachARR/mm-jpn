# @EntitiesInRing (リング内エンティティターゲッター)

## 概要
キャスターを中心としたリング（ドーナツ形）の範囲内にいるすべてのエンティティをターゲットにします。最小距離と最大距離の間にいるエンティティが対象です。

- **タイプ**: 複数エンティティターゲッター
- **省略形**: `@EIRR`

## 属性
| 属性 | 省略形 | 説明 | デフォルト |
|------|--------|------|-----------|
| minrange | min | リングの最小半径（ブロック数） | 5 |
| maxrange | max | リングの最大半径（ブロック数） | 10 |

## 使用例
```yaml
ExampleSkill:
  Skills:
  - ignite @EIRR{min=2;max=10}  # キャスターから2〜10ブロックのリング内のエンティティに火をつける
```

## 備考
- 特定の距離範囲内のエンティティだけをターゲットにする場合に便利です

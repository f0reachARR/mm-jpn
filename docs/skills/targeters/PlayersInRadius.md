# @PlayersInRadius (半径内プレイヤーターゲッター)

## 概要
キャスターを中心とした指定半径内のすべてのプレイヤーをターゲットにします。

- **タイプ**: 複数エンティティターゲッター
- **省略形**: `@PIR`

## 属性
| 属性 | 省略形 | 説明 | デフォルト |
|------|--------|------|-----------|
| radius | r | ターゲッターの半径（ブロック数） | 5 |

## 使用例
```yaml
ExampleSkill:
  Skills:
  - ignite @PIR{r=10}  # 半径10ブロック内のすべてのプレイヤーに火をつける
```

## 備考
- プレイヤーのみをターゲットにします（モブは含みません）
- エリア攻撃やバフ効果に適しています

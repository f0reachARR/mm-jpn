# @LivingInCone (コーン内エンティティターゲッター)

## 概要
指定した角度・長さ・回転でキャスターの前方にコーン（円錐）を形成し、その中にいるすべての生きているエンティティをターゲットにします。

- **タイプ**: 複数エンティティターゲッター
- **省略形**: `@entitiesInCone`, `@livingEntitiesInCone`, `@LEIC`, `@EIC`

## 属性
| 属性 | 省略形 | 説明 | デフォルト |
|------|--------|------|-----------|
| angle | a | コーンの角度（度） | 90 |
| range | r | コーンの長さ（ブロック数） | 16 |
| rotation | rot | コーンの回転（度） | 0 |
| usepitch | pitch, p | ピッチに基づいてコーンを生成するか | false |
| yoffset | yo | コーンのY軸オフセット | 0 |
| livingonly | lo | 生きているエンティティのみをターゲットにするか | true |

## 使用例
```yaml
ExampleSkill:
  Skills:
  - ignite @LivingInCone{a=45;r=20}  # 前方45度、20ブロック先のコーン内のエンティティに火をつける
```

## 備考
- 扇形攻撃パターンの実装に適しています
- `angle=360` で全方向をターゲットにできます

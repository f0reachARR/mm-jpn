# @NotLivingNearOrigin (原点付近の非生物ターゲッター)

## 概要
メタスキルの原点付近にいる非生物エンティティ（アイテム、矢など）をすべてターゲットにします。

- **タイプ**: 複数エンティティターゲッター
- **省略形**: `@nonLivingNearOrigin`, `@NLNO`

## 属性
| 属性 | 省略形 | 説明 | デフォルト |
|------|--------|------|-----------|
| radius | r | ターゲッターの半径（ブロック数） | 5 |

## 使用例
```yaml
ExampleSkill:
  Skills:
  - projectile{...;
    onEnd=[
      - command{c="say <target.uuid>"} @NotLivingNearOrigin{r=10}
      # 終了時、原点から10ブロック以内の非生物エンティティのUUIDを表示
    ]}
```

## 備考
- 非生物エンティティ（ドロップアイテム、矢、ボートなど）をターゲットにします
- プロジェクタイルの `onEnd` メタスキルと組み合わせると効果的です

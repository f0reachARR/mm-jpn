# @Pin (ピン位置ターゲッター)

## 概要
ピン（Pin）の位置をターゲットにします。ピンワンドで作成されたマルチピンの場合、関連するすべての位置をターゲットにします。

- **タイプ**: 複数場所ターゲッター

## 属性
| 属性 | 省略形 | 説明 | デフォルト |
|------|--------|------|-----------|
| name | n, pin, p, key, k | ターゲットとするピンの名前 | |
| random | r | マルチピンの場合、ランダムに1つの位置を選択するか | false |

## 使用例
```yaml
ExampleSkill:
  Skills:
  - teleport @Pin{p=mypin}  # 「mypin」という名前のピン位置にテレポート
```

## 備考
- ピン（Pin）機能については [Pins](/Pins) を参照してください
- マルチピンを使用すると複数の固定位置をまとめてターゲットにできます

# @Children (子エンティティターゲッター)

## 概要
キャスターが召喚した子エンティティをターゲットにします。

- **タイプ**: 複数エンティティターゲッター
- **省略形**: `@child`, `@summons`

## 属性
> *このターゲッターには属性がありません*

## 使用例
```yaml
ExampleSkill:
  Skills:
  - heal{a=10} @Children  # すべての子エンティティを回復
```

## 備考
- `summon` メカニクスで召喚されたエンティティが対象です
- プレイヤーキャスターの場合、`summon` メカニクスで `summonerIsOwner=true` を設定した場合に機能します
- `@Parent` ターゲッターの逆の関係です

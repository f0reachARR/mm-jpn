# @ItemsInRadius (半径内アイテムターゲッター)

## 概要
キャスターを中心とした指定半径内のすべてのアイテム（ドロップされたアイテムエンティティ）をターゲットにします。

- **タイプ**: 複数エンティティターゲッター
- **省略形**: `@IIR`

## 属性
| 属性 | 省略形 | 説明 | デフォルト |
|------|--------|------|-----------|
| radius | r | ターゲッターの半径（ブロック数） | 5 |

## 使用例
```yaml
ExampleSkill:
  Skills:
  - pickupitem @ItemsInRadius{r=10}  # 半径10ブロック内のアイテムをすべて拾う
```

## 備考
- ドロップされたアイテムエンティティをターゲットにします
- `pickupitem` メカニクスと組み合わせて使うことが多いです

# @Sphere (球形状位置ターゲッター)

## 概要
キャスターを中心とした球体上の位置をターゲットにします。

- **タイプ**: 複数場所ターゲッター

## 属性
| 属性 | 省略形 | 説明 | デフォルト |
|------|--------|------|-----------|
| radius | r | 球の半径（ブロック数） | 5 |
| points | p | ポイントの数 | 10 |
| yoffset | y | Y軸のオフセット | 0 |
| exact | e | 完全な球を描画するか | false |

## 使用例
```yaml
ExampleSkill:
  Skills:
  - effect:particles @Sphere{r=6;points=50}  # 半径6ブロックの球上に50個のパーティクル
```

## 備考
- `exact=true` でより均一な球体を生成します
- 球状の爆発エフェクトやバリアに適しています

# @Passenger (パッセンジャーターゲッター)

## 概要
キャスターモブに乗っているライダー（エンティティ）をターゲットにします。

- **タイプ**: 単一エンティティターゲッター
- **省略形**: `@rider`

## 属性
> *このターゲッターには属性がありません*

## 使用例
```yaml
ExampleSkill:
  Skills:
  - message{m="Get off!"} @Passenger  # 乗っているエンティティにメッセージを送信
```

## 備考
- キャスターモブの上に乗っているエンティティをターゲットにします
- `@Vehicle` ターゲッターの逆の関係です

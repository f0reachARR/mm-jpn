# SendResourcePack（リソースパック送信）

## 説明

ターゲットのプレイヤーにリソースパックを送信します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| url | u, link | \1の名前 | NONE |
| hash | h | \1の名前 | mythicmobs |

## エイリアス

- [x] resourcepack

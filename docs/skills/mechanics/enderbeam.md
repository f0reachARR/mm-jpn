# EnderBeam（エンダービーム）

## 説明

起点からターゲットへエンダークリスタルのビームエフェクトを作成します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| duration | d | ビームの継続時間（tick単位） | 1 |

## 使用例

```yaml
  Skills:
  - enderbeam{d=20} @target ~onTimer:5
```

## エイリアス
- [x] crystalbeam
- [x] effect:enderbeam

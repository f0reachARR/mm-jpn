# BlockDestabilize（ブロック不安定化）

## 説明

ターゲットのブロックを重力の影響を受けたように落下させます。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| radius | r | 不安定化するブロックの範囲 | 1 |

## 使用例

```yaml
  Skills:
  - blockdestabilize{r=3} @target ~onTimer:20
```

## エイリアス
- [x] destabilizeblock
- [x] blockunstable

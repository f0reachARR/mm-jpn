# FakeLightning（フェイク雷撃）

## 説明

ターゲットにフェイクの雷を落とします（ダメージなし）。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| localized | l | Whether the lightning should only be seen/heard by players in radius | false |
| localizedradius | lr, r | ターゲット周囲の半径 | 128 |

## エイリアス

- [x] effect:lightning
- [x] e:lightning

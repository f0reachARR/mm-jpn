# SetMaxHealth（最大体力設定）

## 説明

ターゲットエンティティの最大体力を設定します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| amount | a | 変更量 | 1.0 |
| mode | m | \1の名前 | STATIC<!--type:SetMaxHealth_Mode--> |

## エイリアス

- [x] setmaxhp

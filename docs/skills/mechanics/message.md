# Message（メッセージ送信）

## 説明

ターゲットのプレイヤーにメッセージを送信します。[カラーコード](/Skills/Placeholders#color-codes)と[変数](/Skills/Variables)が使用可能です。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| message | msg,m | 送信するメッセージ | None |
| audience |  | メッセージの[オーディエンス] | <!--type:Audience--> |

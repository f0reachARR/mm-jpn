# ParticleSphere（パーティクル球体）

## 説明

ターゲットの周囲にパーティクルの球体を描きます。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| radius | r | ターゲット周囲の半径 | 0 |

## エイリアス

- [x] effect:particlesphere
- [x] e:ps
- [x] ps

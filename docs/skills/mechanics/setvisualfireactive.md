# SetVisualFireActive（視覚的炎設定）

## 説明

ターゲットエンティティの視覚的な炎エフェクト（実際にダメージを与えない見た目だけの炎）を有効/無効にします。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| active | a | 視覚的炎を有効にするか | true |

## 使用例

```yaml
  Skills:
  - setvisualfireactive{active=true} @target
```

# Hologram（ホログラム）

## 説明

ターゲット位置にホログラムを召喚します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| text | t | 表示するテキスト | Hologram Text Missing |
| stay | time, staytime | 持続時間（tick） | 100 |

# PushBlock（ブロック押し出し）

## 説明

ターゲット位置のブロックを指定された方向に押し出します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| direction | dir, d | \1の名前 | NORTH<!--type:NORTH,SOUTH,EAST,WEST,UP,DOWN--> |
| onPush | then | 実行するメタスキル | <!--type:Metaskill--> |
| onFail |  | 実行するメタスキル | <!--type:Metaskill--> |

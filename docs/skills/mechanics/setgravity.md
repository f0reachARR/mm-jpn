# SetGravity（重力設定）

## 説明

重力がターゲットエンティティに影響するかどうかを設定します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| gravity | g, b, bool, u, use | Sets whether the entity uses gravity | true |

## エイリアス

- [x] setusegravity

# Shockwave（衝撃波）

## 説明

衝撃波エフェクトを作成します。近くのエンティティを吹き飛ばします。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| radius | r | 衝撃波の半径 | 3 |
| knockback | kb | ノックバックの強さ | 2 |
| damage | d | 与えるダメージ量 | 0 |

## 使用例

```yaml
  Skills:
  - shockwave{radius=5;knockback=3} @self ~onDamaged
```

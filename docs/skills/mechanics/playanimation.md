# PlayAnimation（アニメーション再生）

## 説明

エンティティにアニメーションを強制的に再生させます。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| animation | a, effect, e | 再生するアニメーション | 1 |
| audience |  | メッセージの[オーディエンス] | world<!--type:Audience--> |

## エイリアス

- [x] effect:playanimation
- [x] e:playanimation
- [x] playarmanimation

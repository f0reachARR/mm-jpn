# Weather（天候変更）

## 説明

天候を変更します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| type | t | \1の名前 | sunny<!--type:SUNNY,RAINY,STORMY--> |
| duration | d | How long (in ticks) the weather will be forced to last | 500 |

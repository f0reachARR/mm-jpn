# Steer（操縦）

## 説明

キャスターを操縦します。騎乗しているプレイヤーがモブを操縦できるかどうかを設定します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| forward | f | 前方への力 | 0 |
| sideways | s | 横方向への力 | 0 |

## 使用例

```yaml
  Skills:
  - steer @self ~onTimer:1
```

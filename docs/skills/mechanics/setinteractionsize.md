# SetInteractionSize（インタラクションサイズ設定）

## 説明

インタラクションエンティティのサイズを設定します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| height | h | 設定するスコア値 | 1 |
| width | w | 設定するスコア値 | 1 |

## エイリアス

- [x] interactionSize

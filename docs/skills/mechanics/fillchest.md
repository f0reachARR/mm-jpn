# FillChest（チェスト充填）

## 説明

チェストにアイテム、またはドロップテーブルのアイテムを充填します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| item | i | 充填するアイテムまたはドロップテーブル名 | |
| amount | a | アイテムの数量 | 1 |
| usedroptable | udt | ドロップテーブルを使用するか | false |
| droptable | dt | 使用するドロップテーブル名 | |

## 使用例

```yaml
  Skills:
  - fillchest{item=DIAMOND;amount=5} @targetblock ~onInteract
```

## エイリアス
- [x] fc
- [x] chest

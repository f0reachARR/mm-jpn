# BoneMeal（骨粉エフェクト）

## 説明

ターゲットのブロックに骨粉エフェクトを適用します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| amount | a | 骨粉を適用する回数 | 1 |

## 使用例

```yaml
  Skills:
  - bonemeal{amount=3} @target ~onTimer:20
```

## エイリアス
- [x] bm
- [x] applybonemeal

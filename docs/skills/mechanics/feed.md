# Feed（食事）

## 説明

ターゲットプレイヤーの食料レベルを回復させます。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| amount | a | 回復する食料量 | 1 |
| saturation | s | 回復する満腹度 | 0 |

## 使用例

```yaml
  Skills:
  - feed{amount=4;saturation=4} @trigger ~onInteract
```

## エイリアス
- [x] f

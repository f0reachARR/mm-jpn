# Recoil（リコイル）

## 説明

リコイルをシミュレートするためにターゲットの画面を揺らします。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| recoil | r | 量 | 1 |
| pitch | p | \1の名前 | -10to10 |
| yaw | y | \1の名前 | -0.25to0.25 |
| sneaking | sn | \1の名前 | 0.75 |
| sprinting | sp | \1の名前 | 1.25 |

## エイリアス

- [x] effect:recoil
- [x] e:recoil

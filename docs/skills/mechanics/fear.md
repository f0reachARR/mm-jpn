# Fear（恐怖）

## 説明

ターゲットを恐怖で逃げ回らせるオーラを適用します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| duration | d | オーラの継続時間（tick単位） | 100 |
| interval | i | 逃げる動作の更新間隔（tick単位） | 5 |

## 使用例

```yaml
  Skills:
  - fear{d=60} @target ~onDamaged
```

## エイリアス
- [x] scare

# Geyser（間欠泉）

## 説明

水または溶岩の「間欠泉」を作成します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| material | m, fluid | 使用する流体の種類（`WATER` または `LAVA`） | WATER |
| height | h | 間欠泉の高さ | 5 |
| radius | r | 間欠泉の範囲 | 1 |
| duration | d | 継続時間（tick単位） | 20 |
| damage | dmg | 間欠泉によるダメージ量 | 0 |

## 使用例

```yaml
  Skills:
  - geyser{m=WATER;h=8;r=2;d=40} @targetlocation ~onTimer:20
```

## エイリアス
- [x] g

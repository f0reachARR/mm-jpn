# OpenTrades（取引メニューを開く）

## 説明

ターゲットのプレイヤーにキャストしているビレッジャーの取引メニューを開きます。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| realTrade | real | 開くか（falseで閉じる） | true |

## エイリアス

- [x] opentrade
- [x] trade

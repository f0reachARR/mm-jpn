# Fly（飛行）

## 説明

ターゲットプレイヤーが飛行できるようにするオーラを適用します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| duration | d | オーラの継続時間（tick単位） | 100 |

## 使用例

```yaml
  Skills:
  - fly{d=200} @trigger ~onInteract
```

## エイリアス
- [x] enablefly
- [x] allowfly

# PushButton（ボタン押下）

## 説明

ターゲット位置のボタンを押します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| x |  | \1の名前 | 0 |
| y |  | \1の名前 | 0 |
| z |  | \1の名前 | 0 |
| location | loc, l | Location of the action, in a `x,y,z` syntax. If set, other attributes are ignored |  |

## エイリアス

- [x] buttonpush

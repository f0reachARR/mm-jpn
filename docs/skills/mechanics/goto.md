# GoTo（移動）

## 説明

ターゲターの位置（エンティティまたは場所）に向かって移動します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| speed | s, v, velocity | 移動速度 | 1 |
| tolerance | t | 到達とみなす距離の許容誤差 | 1 |
| maxDistance | md | 最大移動距離 | 50 |

## 使用例

```yaml
  Skills:
  - goto{speed=1.5} @target ~onTimer:5
```

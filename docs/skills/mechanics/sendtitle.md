# SendTitle（タイトルメッセージ送信）

## 説明

ターゲットのプレイヤーにタイトル/サブタイトルメッセージを送信します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| title | t | \1の名前 |  |
| subtitle | st | \1の名前 |  |
| duration | d | How long the title will display (in ticks) | 1 |
| fadeIn | fi | フェードイン時間（tick） | 1 |
| fadeOut | fo | フェードイン時間（tick） | 1 |

## エイリアス

- [x] title

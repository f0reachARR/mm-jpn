# ToggleLever（レバー切替）

## 説明

ターゲット位置のレバーを切り替えます。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| location | loc, l | Location of the action, in a `x,y,z` syntax. If set, other location attributes are ignored |  |
| duration | d | 持続時間（tick） | 20 |
| x |  | \1の名前 | 0 |
| y |  | \1の名前 | 0 |
| z |  | \1の名前 | 0 |

## エイリアス

- [x] lever

# SetLevel（レベル設定）

## 説明

キャストするモブのレベルを変更します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| action | a | \1の名前 | SET<!--type:SET,ADD,SUBTRACT,MULTIPLY,DIVIDE--> |
| level | l | Amount used in the operation | 1 |

## エイリアス

- [x] modifylevel

# SetAI（AI 設定）

## 説明

ターゲットモブのAIを無効/有効にします。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| ai | state, value | Sets the new mob AI. `true` enables it, `false` disables it | false |

## エイリアス

- [x] ai

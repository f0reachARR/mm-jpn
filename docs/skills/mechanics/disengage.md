# Disengage（後退）

## 説明

キャスターをターゲットエンティティから後方に跳び退かせます。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| velocity | v | 跳び退く速度 | 1 |

## 使用例

```yaml
  Skills:
  - disengage{v=2} @target ~onDamaged
```

## エイリアス
- [x] dis
- [x] leap

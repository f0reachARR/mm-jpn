# SetRotation（回転設定）

## 説明

ターゲットの回転を設定します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| relative | r, rel | If the change is relative to the target, boolean | false |
| yaw | y | \1の名前 | 0 |
| pitch | p | \1の名前 | 0 |

## エイリアス

- [x] setrot

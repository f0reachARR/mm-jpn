# BarRemove（ボスバー削除）

## 説明

キャストしているモブのカスタムボスバーを削除します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| name | n | 削除するボスバーの識別名 | |

## 使用例

```yaml
  Skills:
  - barremove{name=mybar} @self ~onDeath
```

## エイリアス
- [x] bossbarremove
- [x] removebar

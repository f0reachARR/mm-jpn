# HealPercent（割合回復）

## 説明

ターゲットエンティティを最大体力の指定した割合で回復します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| multiplier | m, amount, a | \1の名前 | 0.1 |
| overheal | oh | オーバーヒールを最大体力の追加として適用するか | false |
| maxoverheal | maxabsorb, maxshield, mo, ma, ms | 最大オーバーヒール量 | 0 |
| tags | tag | 任意のタグを指定できます（カンマ区切り、大文字化される） |  |
| rawtags | rtag | tagsと同様ですが大文字化されません |  |

## エイリアス

- [x] percentheal
- [x] hp

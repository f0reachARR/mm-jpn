# Heal（回復）

## 説明

ターゲットエンティティを指定された値で回復します。最大体力を超えて「オーバーヒール」することも可能です。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| amount | a | 回復量 | 1 |
| overheal | oh | オーバーヒールを最大体力の追加として適用するか | false |
| maxoverheal | maxabsorb, maxshield, mo, ma, ms | 最大オーバーヒール量 | 1 |
| tags | tag | 任意のタグを指定できます（カンマ区切り、大文字化される） |  |
| rawtags | rtag | tagsと同様ですが大文字化されません |  |

## エイリアス

- [x] h

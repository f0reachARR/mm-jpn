# ParticleLineHelix（パーティクルヘリックス）

## 説明

ラインベースのヘリックス（螺旋）エフェクトを描きます。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| distanceBetween | db | \1の名前 | 1 |
| startYOffset | syo, ystartoffset, ys | Offset Y location of the starting point | 0 |
| targetYOffset | tyo, ytargetoffset, yt | Offset Y location of the target point | 0 |
| fromOrigin | fo | するかどうか | false |
| helixlength | hl | 長さ | 2 |
| helixradius | hr | ターゲット周囲の半径 | 1 |
| helixrotation | rot | \1の名前 | 0 |
| maxdistance | md | \1の名前 | 256 |

## エイリアス

- [x] effect:particlelinehelix
- [x] particlehelixline

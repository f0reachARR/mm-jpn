# Glow（グロー）

## 説明

ターゲットを発光（グロー）させます。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| duration | d | 発光の継続時間（tick単位） | 100 |
| color | c | 発光の色（チームカラー） | WHITE |

## 使用例

```yaml
  Skills:
  - glow{d=200;color=RED} @target ~onDamaged
```

## エイリアス
- [x] g
- [x] addglow

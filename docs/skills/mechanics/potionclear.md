# PotionClear（ポーション効果解除）

## 説明

ターゲットエンティティからすべてのポーション効果を除去します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| types | type, t | \1の名前 | <!--type:PotionEffectType--> |

## エイリアス

- [x] clearpotion
- [x] clearpotions

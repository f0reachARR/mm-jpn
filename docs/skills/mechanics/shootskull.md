# ShootSkull（スカル射出）

## 説明

スカル（エンドラの頭部）を射出します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| yield | strength, y, s | \1の名前 | 1 |
| playsound | ps | するかどうか | false |

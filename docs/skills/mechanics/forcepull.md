# ForcePull（強制引き寄せ）

## 説明

ターゲットをキャスターの位置にテレポートさせます。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| pull | p | 引き寄せの強さ | 1 |

## 使用例

```yaml
  Skills:
  - forcepull @target ~onTimer:20
```

## エイリアス
- [x] pull
- [x] fp

# Log（コンソールログ）

## 説明

コンソールにメッセージを記録します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| message | m | \1の名前 | "" |

## エイリアス

- [x] console
- [x] print
- [x] printf

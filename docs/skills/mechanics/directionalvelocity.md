# DirectionalVelocity（方向速度）

## 説明

ターゲットエンティティの特定のベクトル方向の速度を変更します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| forward | f | 前方向の速度 | 0 |
| sideways | s, side | 横方向の速度 | 0 |
| upward | u, up | 上方向の速度 | 0 |

## 使用例

```yaml
  Skills:
  - directionalvelocity{f=1;u=0.5} @trigger ~onDamaged
```

## エイリアス
- [x] dv
- [x] dirvel

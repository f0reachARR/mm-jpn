# Freeze（凍結）

## 説明

パウダースノーの凍結エフェクトを使用して、指定したtick数の間ターゲットを凍結させます。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| duration | d, ticks | 凍結の継続時間（tick単位） | 100 |
| particle | p | 凍結パーティクルを表示するか | true |

## 使用例

```yaml
  Skills:
  - freeze{d=100} @target ~onTimer:20
```

## エイリアス
- [x] ice
- [x] frost

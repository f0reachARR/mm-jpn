# Saddle（鞍装備）

## 説明

ターゲットエンティティに鞍を装備または取り外します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| take | t, remove, r | Whether the saddle should be taken away instead | false |

## エイリアス

- [x] setsaddle
- [x] givesaddle

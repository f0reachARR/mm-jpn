# BossBorder（ボスボーダー）

## 説明

モブの周囲に逃げられないボーダーを作成します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| radius | r | ボーダーの半径 | 30 |
| duration | d | ボーダーの継続時間（tick単位） | 0 |

## 使用例

```yaml
  Skills:
  - bossborder{r=25;d=6000} @self ~onSpawn
```

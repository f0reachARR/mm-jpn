# GiveItemFromTarget（ターゲットからアイテム付与）

## 説明

ターゲットエンティティまたは位置からピックアップアニメーションを再生しながら、キャスターにアイテムを付与します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| item | i | 付与するアイテム名 | |
| amount | a | アイテムの数量 | 1 |

## 使用例

```yaml
  Skills:
  - giveitemfromtarget{item=DIAMOND;amount=2} @target ~onTimer:20
```

## エイリアス
- [x] gift
- [x] giveitemtarget

# CurrencyGive（通貨付与）

## 説明

プレイヤーにお金を付与します。Vault および通貨プラグインが必要です。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| amount | a | 付与する金額 | 1 |

## 使用例

```yaml
  Skills:
  - currencygive{amount=100} @trigger ~onInteract
```

## エイリアス
- [x] givemoney
- [x] givecurrency

# ParticleBox（パーティクルボックス）

## 説明

ターゲットの周囲にパーティクルのボックスを描きます。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| radius | r | ターゲット周囲の半径 | 5 |

## エイリアス

- [x] effect:particlebox
- [x] e:pb
- [x] pb

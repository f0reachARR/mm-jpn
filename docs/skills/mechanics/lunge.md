# Lunge（突進）

## 説明

キャスターをターゲットに向かって突進させます。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| velocity | v, magnitude | \1の名前 | 1 |
| velocityY | vy, yv, yvelocity | \1の名前 | 1 |
| oldmath | old, o | If the lunge mechanic should use the old math formula | false |

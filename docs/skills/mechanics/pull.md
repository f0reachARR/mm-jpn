# Pull（引き寄せ）

## 説明

ターゲットをモブの方向に引き寄せます。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| velocity | v | \1の名前 | 1 |
| toOrigin | to | Wether or not the target should pulled towards the origin of the skill | false |

# BloodyScreen（血まみれの画面）

## 説明

ターゲットの画面を赤く輝かせます（ダメージエフェクト）。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| duration | d | エフェクトの継続時間（tick単位） | 20 |
| damage | dmg | ダメージ量（画面の赤みを制御） | 1 |

## 使用例

```yaml
  Skills:
  - bloodyscreen{d=40;damage=10} @target ~onTimer:20
```

## エイリアス
- [x] bs
- [x] redscreen

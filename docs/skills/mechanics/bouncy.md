# Bouncy（バウンシー）

## 説明

ターゲットをバウンシー（弾む）状態にするオーラを適用します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| duration | d | オーラの継続時間（tick単位） | 100 |
| bouncemultiplier | bmult, bm | バウンスの倍率 | 1 |

## 使用例

```yaml
  Skills:
  - bouncy{d=200;bm=2} @target ~onTimer:20
```

## エイリアス
- [x] bounce

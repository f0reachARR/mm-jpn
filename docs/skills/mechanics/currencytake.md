# CurrencyTake（通貨剥奪）

## 説明

プレイヤーからお金を剥奪します。Vault および通貨プラグインが必要です。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| amount | a | 剥奪する金額 | 1 |

## 使用例

```yaml
  Skills:
  - currencytake{amount=50} @trigger ~onInteract
```

## エイリアス
- [x] takemoney
- [x] takecurrency

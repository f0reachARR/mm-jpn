# BlockUnmask（ブロックマスク解除）

## 説明

マスクされたブロックのマスクを解除します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| radius | r | マスクを解除するブロックの範囲 | 0 |

## 使用例

```yaml
  Skills:
  - effect:blockunmask{r=30} @self ~onTimer:20
```

## エイリアス
- [x] effect:blockUnmask
- [x] e:blockUnmask

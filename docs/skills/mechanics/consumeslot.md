# ConsumeSlot（スロットアイテム消費）

## 説明

プレイヤーのインベントリの指定スロットからアイテムを削除します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| slot | s | アイテムを削除するスロット番号 | 0 |
| amount | a | 削除するアイテムの数量 | 1 |

## 使用例

```yaml
  Skills:
  - consumeslot{slot=0;amount=1} @trigger ~onInteract
```

## エイリアス
- [x] cs
- [x] consumeitem

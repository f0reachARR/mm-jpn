# ParticleRing（パーティクルリング）

## 説明

ターゲットの周囲にパーティクルのリングを描きます。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| points | pts | \1の名前 | 8 |
| radius | r | ターゲット周囲の半径 | 10 |

## エイリアス

- [x] effect:particlering
- [x] e:pr

# Doppleganger（ドッペルゲンガー）

## 説明

ターゲットプレイヤーの外見をコピーします。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| duration | d | 変装の継続時間（tick単位） | 200 |

## 使用例

```yaml
  Skills:
  - doppleganger{d=400} @trigger ~onInteract
```

## エイリアス
- [x] doppelganger

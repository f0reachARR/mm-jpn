# VSkill（ビジュアルスキル）

## 説明

ビジュアルスキルを実行します。エフェクトのみのスキルを実行するために使用されます。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| skill | s | 実行するスキルの名前 | |

## 使用例

```yaml
  Skills:
  - vskill{skill=MyVisualSkill} @self ~onTimer:20
```

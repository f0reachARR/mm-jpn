# Windup（ウィンドアップ）

## 説明

メカニックを実行する前にウィンドアップエフェクトを作成します。攻撃予告やアニメーションの前置きとして使用されます。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| duration | d | ウィンドアップの持続時間（tick） | 20 |
| startskill | start, ss | ウィンドアップ開始時に実行するスキル | |
| endskill | end, es | ウィンドアップ終了時に実行するスキル | |
| cancelskill | cancel, cs | ウィンドアップがキャンセルされた時に実行するスキル | |
| particle | p | ウィンドアップ中に表示するパーティクル | |

## 使用例

```yaml
  Skills:
  - windup{duration=40;startskill=WindupStart;endskill=WindupEnd} @self
```

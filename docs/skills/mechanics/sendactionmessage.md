# SendActionMessage（アクションバーメッセージ送信）

## 説明

ターゲットのプレイヤーにアクションバーメッセージを送信します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| message | m, msg | 送信するメッセージ |  |

## エイリアス

- [x] am
- [x] actionmessage

# BlackScreen（画面暗転）

## 説明

指定した時間、ターゲットプレイヤーの画面を暗転させます（盲目エフェクト）。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| duration | d | 暗転の継続時間（tick単位） | 20 |

## 使用例

```yaml
  Skills:
  - blackscreen{d=60} @target ~onDamaged
```

## エイリアス
- [x] bs

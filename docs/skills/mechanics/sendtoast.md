# SendToast（トースト通知送信）

## 説明

ターゲットのプレイヤーに実績トーストを送信します。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| icon | i | \1の名前 | diamond_sword<!--type:Item--> |
| iconnbt | nbt | \1の名前 | NONE |
| message | msg,m | 送信するメッセージ | NONE |
| frame | f | \1の名前 | challenge<!--type:AdvancementDisplayFrame--> |

## エイリアス

- [x] toastmessage
- [x] toastmsg
- [x] advmessage
- [x] advancementmessage

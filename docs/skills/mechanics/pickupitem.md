# PickupItem（アイテム拾得）

## 説明

ターゲットのアイテムを拾います。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| fakelooting | fl | Whether a fake looting animation should be played | true |
| onpickup | pickup, then | The [metaskill] to execute once the item is picked up | <!--type:Metaskill--> |

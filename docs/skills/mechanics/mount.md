# Mount（乗騎召喚）

## 説明

キャスターのためにモブを召喚し、それに乗ります。

## 属性

| 属性 | エイリアス | 説明 | デフォルト値 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| type | t, mob, m | \1の名前 | <!--type:Mob--> |
| stack | s | If the summoned mob should stack atop existing mounted entities | false |

## エイリアス

- [x] vehicle

# プレースホルダー (Placeholders)

これらはスキルや文字列を使用するメカニクスで使用できるすべてのプレースホルダーと特殊文字です。

> **注意**: スキルでの多くの変数の使用はプレミアムバージョンに制限されています（MMOCore、特殊文字、カラーコード、`setvariable`を除く）。

## 特殊文字

| プレースホルダー | 機能 | 記号 |
|:------------:|------|------|
| `<&co>` | コロンを返します | `:` |
| `<&sq>` | アポストロフィを返します | `'` |
| `<&da>` | ダッシュを返します | `-` |
| `<&bs>` | バックスラッシュを返します | `\` |
| `<&fs>` | スラッシュを返します | `/` |
| `<&sp>` | スペースを返します | ` ` |
| `<&cm>` | カンマを返します | `,` |
| `<&sc>` | セミコロンを返します | `;` |
| `<&eq>` | イコール記号を返します | `=` |
| `<&dq>` | ダブルクォートを返します | `"` |
| `<&rb>` | 右括弧を返します | `]` |
| `<&lb>` | 左括弧を返します | `[` |
| `<&rc>` | 右波括弧を返します | `}` |
| `<&lc>` | 左波括弧を返します | `{` |
| `<&nm>` | ナンバー記号を返します | `#` |
| `<&nl>` | 改行を強制します | |
| `<&heart>` | ハートを返します | `❤` |
| `<&skull>` | 骸骨マークを返します | `☠` |
| `<&lt>` | 小なり記号を返します | `<` |
| `<&gt>` | 大なり記号を返します | `>` |

## カラーコード

これらのカラーコードはモブおよびスキルファイルのどこでも機能します。

| コード | 色 | コード | 色 |
|:----:|------|:----:|------|
| `&0` | 黒 | `&B` | 水色 |
| `&1` | 濃い青 | `&C` | 赤 |
| `&2` | 濃い緑 | `&D` | 薄い紫 |
| `&3` | 濃い水色 | `&E` | 黄色 |
| `&4` | 濃い赤 | `&F` | 白 |
| `&5` | 濃い紫 | `&K` | 魔法 |
| `&6` | 金色 | `&L` | 太字 |
| `&7` | 灰色 | `&M` | 打ち消し線 |
| `&8` | 濃い灰色 | `&N` | 下線 |
| `&9` | 青 | `&O` | イタリック |
| `&A` | 緑 | `&R` | リセット |

[MiniMessageタグ](https://docs.adventure.kyori.net/minimessage/format.html#standard-tags)の使用が推奨されています：

```yml
MessageSkill:
  Skills:
    - message{m=<rainbow>虹色のテキスト</rainbow> でも <red>これは赤</red>！} @target
```

## キャスタープレースホルダー

これらのプレースホルダーはキャスターの属性を返します。

| キャスタープレースホルダー | 機能 |
|:--------------------:|------|
| `<caster.damage>` | キャスターの攻撃ダメージ属性値を返します |
| `<caster.display>` | キャスターの表示名を返します |
| `<caster.mythic_type>` | キャスターの内部モブタイプを返します |
| `<caster.type>` | MythicMobの内部IDまたはエンティティ名を返します |
| `<caster.uuid>` | キャスターのUUIDを返します |
| `<caster.level>` | キャスターのレベルを返します |
| `<caster.name>` | キャスターの名前を返します |
| `<caster.hp>` | キャスターの現在HPを返します |
| `<caster.mhp>` | キャスターの最大HPを返します |
| `<caster.php>` | キャスターのHP割合を返します |
| `<caster.l.w>` | キャスターがいるワールド名を返します |
| `<caster.l.x>` | キャスターのX座標を返します |
| `<caster.l.y>` | キャスターのY座標を返します |
| `<caster.l.z>` | キャスターのZ座標を返します |
| `<caster.l.yaw>` | キャスターのヨーを返します |
| `<caster.l.pitch>` | キャスターのピッチを返します |
| `<caster.stance>` | キャスターの現在のスタンスを返します |
| `<caster.skill.{Metaskill}.cooldown>` | 指定されたスキルの現在のクールダウンを返します |
| `<caster.children.size>` | このエンティティの子の数を返します |

## ターゲットプレースホルダー

| ターゲットプレースホルダー | 機能 |
|:--------------------:|------|
| `<target.uuid>` | ターゲットのUUIDを返します |
| `<target.name>` | ターゲットの名前を返します |
| `<target.hp>` | ターゲットの現在HPを返します |
| `<target.mhp>` | ターゲットの最大HPを返します |
| `<target.php>` | ターゲットのHP割合を返します |
| `<target.l.x>` | ターゲットのX座標を返します |
| `<target.l.y>` | ターゲットのY座標を返します |
| `<target.l.z>` | ターゲットのZ座標を返します |
| `<target.l.w>` | ターゲットがいるワールドを返します |
| `<target.level>` | ターゲットのレベルを返します |

## 変数プレースホルダー

変数には特定のプレースホルダー構文でアクセスできます：

```
<caster.var.variable_name>    # キャスターの変数
<target.var.variable_name>    # ターゲットの変数
<skill.var.variable_name>     # スキルスコープの変数
<global.var.variable_name>    # グローバル変数
```

## ランダム生成

```
<random.#to#>           # 整数のランダムな数値
<random.float.#to#>     # フロートのランダムな数値
```

## ユーティリティプレースホルダー

```
<utils.epoch.timestamp>   # 現在のエポック時刻
<caster.raytrace.{Float}> # キャスターが見ているブロック名
```

## 使用例

```yaml
ExampleSkill:
  Skills:
  - message{m="こんにちは、<target.name>！あなたのHPは<target.hp>/<target.mhp>です。"} @PlayersInRadius{r=10}
  - message{m="私の現在の座標: X<caster.l.x> Y<caster.l.y> Z<caster.l.z>"} @self
  - message{m="ランダムな数字: <random.1to100>"} @self
```

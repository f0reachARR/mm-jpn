# hasaura

**タイプ**: 🟦 エンティティ (Entity)

## 説明

ターゲットエンティティが指定されたオーラを持っているかどうかをチェックします。

> *Checks if the target entity has the given aura*

## 使用例

```yaml
  Conditions:
  - hasaura{aura=firedebuff} true
```

## エイリアス

- `hasbuff`
- `hasdebuff`

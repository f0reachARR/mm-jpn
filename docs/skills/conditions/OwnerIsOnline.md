# OwnerIsOnline

**タイプ**: 🟦 エンティティ (Entity)

## 説明

エンティティのオーナーがオンラインかどうかをチェックします。

> *Checks if the owner of the target mob is online*

## 使用例

```yaml
  Conditions:
  - ownerisonline true
```

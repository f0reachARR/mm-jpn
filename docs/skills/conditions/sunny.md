# sunny

**タイプ**: 🟩 ロケーション (Location)

## 説明

天気が晴れかどうかをチェックします。

> *Checks if the weather is sunny in the target world.*

## 使用例

```yaml
  Conditions:
  - sunny true
```

## エイリアス

- `issunny`

# isMythicMob

**タイプ**: 🟦 エンティティ (Entity)

## 説明

ターゲットがMythicMobかどうかをテストします。

> *Checks if the target entity is a MythicMob*

## 使用例

```yaml
Conditions:
- isMythicMob true
```

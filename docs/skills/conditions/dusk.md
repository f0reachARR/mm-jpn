# dusk

**タイプ**: 🟩 ロケーション (Location)

## 説明

時刻が夕暮れかどうかをチェックします。

> *Checks if the time in the world at the target location is dusk, from 14000 to 18000 in-game time.*

## 使用例

```yaml
  Conditions:
  - dusk true
```

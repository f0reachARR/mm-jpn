# height

**タイプ**: 🟩 ロケーション (Location)

## 説明

ターゲットのY座標が指定された範囲内かどうかをチェックします。

> *Checks if the target's Y location is within a range*

## 属性

| 属性 | エイリアス | 説明 | デフォルト |
|------|-----------|------|-----------|
| `height` | h | The range of the Y location of the target |  |

## 使用例

```yaml
  Conditions:
  - height{h=<50} true
```

# hasPassenger

**タイプ**: 🟦 エンティティ (Entity)

## 説明

エンティティにパッセンジャーが乗っているかどうかをチェックします。

> *Checks if the target entity has a passenger*

## 使用例

```yaml
  Conditions:
  - hasPassenger true
```

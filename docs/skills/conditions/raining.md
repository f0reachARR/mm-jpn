# raining

**タイプ**: 🟩 ロケーション (Location)

## 説明

ワールドで雨が降っているかどうかをテストします。

> *Checks if it's raining in the target world.*

## 使用例

```yaml
  Conditions:
  - raining true
```

## エイリアス

- `israining`

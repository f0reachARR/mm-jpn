# burning

**タイプ**: 🟦 エンティティ (Entity)

## 説明

ターゲットエンティティが炎上しているかどうかをチェックします。

> *Checks if the target entity is burning*

## 使用例

```yaml
  TargetConditions:
  - isburning true
```

## エイリアス

- `isburning`
- `isonfire`

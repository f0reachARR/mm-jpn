# TargetWithin

**タイプ**: 🟧 比較 (Compare)

## 説明

ターゲットが指定された距離内にいるかどうかをチェックします。

> *Tests if the target's vanilla target is within a certain distance.*

## 属性

| 属性 | エイリアス | 説明 | デフォルト |
|------|-----------|------|-----------|
| `distance` | d | Distance to check | 0 |

## 使用例

```yaml
Conditions:
- TargetWithin true
```

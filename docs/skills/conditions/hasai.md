# hasai

**タイプ**: 🟦 エンティティ (Entity)

## 説明

エンティティがAIを持っているかどうかをチェックします。

> *Checks if the target entity has its AI enabled*

## 使用例

```yaml
  Conditions:
  - hasAI true
```

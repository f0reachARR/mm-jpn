# hasgravity

**タイプ**: 🟦 エンティティ (Entity)

## 説明

エンティティが重力の影響を受けているかどうかをチェックします。

> *If the target has gravity*

## 使用例

```yaml
Conditions:
- hasgravity true
```

## エイリアス

- `gravity`

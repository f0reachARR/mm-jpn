# enchantingLevel

**タイプ**: 🟦 エンティティ (Entity)

## 説明

エンチャントレベルをチェックします。

> *Checks the target player's experience level*

## 属性

| 属性 | エイリアス | 説明 | デフォルト |
|------|-----------|------|-----------|
| `level` | l | Range of amount of experience level to check for | 0 |

## 使用例

```yaml
  TargetConditions:
  - EnchantingLevel{l=<10} true
```

# gliding

**タイプ**: 🟦 エンティティ (Entity)

## 説明

エンティティがエリトラで滑空しているかどうかをチェックします。

> *If the target entity is gliding*

## 使用例

```yaml
  Conditions:
  - gliding true
```

## エイリアス

- `isgliding`

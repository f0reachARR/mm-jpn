# blocking

**タイプ**: 🟦 エンティティ (Entity)

## 説明

ターゲットプレイヤーがシールドでブロックしているかどうかをテストします。

> *This condition tests if the target player is blocking with a shield.*

## 使用例

```yaml
  Conditions:
  - blocking true
```

## エイリアス

- `isblocking`

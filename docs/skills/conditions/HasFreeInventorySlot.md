# HasFreeInventorySlot

**タイプ**: 🟦 エンティティ (Entity)

## 説明

ターゲットのインベントリに空きスロットがあるかどうかをチェックします。

> *Checks if the evaluated entity has a free inventory slot (at all)*

## 使用例

```yaml
  TriggerConditions:
  - hasfreeinventoryslot
```

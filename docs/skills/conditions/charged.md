# charged

**タイプ**: 🟦 エンティティ (Entity)

## 説明

ターゲットのクリーパーがチャージされているかどうかをチェックします。

> *Checks if the target creeper is charged*

## 使用例

```yaml
Conditions:
- charged true
```

## エイリアス

- `ischarged`
- `creepercharged`

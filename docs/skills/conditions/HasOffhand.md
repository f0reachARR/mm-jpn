# HasOffhand

**タイプ**: 🟦 エンティティ (Entity)

## 説明

ターゲットがオフハンドにアイテムを持っているかどうかをチェックします。

> *Checks if the target entity has something in the offhand*

## 使用例

```yaml
  TargetConditions:
  - hasoffhand true
```

## エイリアス

- `offhand`

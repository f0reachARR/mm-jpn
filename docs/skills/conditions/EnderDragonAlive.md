# EnderDragonAlive

**タイプ**: 🟩 ロケーション (Location)

## 説明

エンダードラゴンが生きているかどうかをチェックします。

> *Checks if there is at least one EnderDragon alive in the world of the targeted location*

## 使用例

```yaml
  Conditions:
  - enderdragonalive true
```

## エイリアス

- `isdragonalive`
- `dragonalive`

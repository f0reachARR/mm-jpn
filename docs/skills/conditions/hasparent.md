# hasparent

**タイプ**: 🟦 エンティティ (Entity)

## 説明

エンティティが親を持っているかどうかをチェックします。

> *This condition checks if the target mob has a parent.*

## 使用例

```yaml
  Conditions:
  - hasparent true
```

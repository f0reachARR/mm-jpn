# name

**タイプ**: 🟦 エンティティ (Entity)

## 説明

エンティティの名前をチェックします。

> *Checks against the name of the target entity*

## 属性

| 属性 | エイリアス | 説明 | デフォルト |
|------|-----------|------|-----------|
| `name` | n | The name to check against | Ashijin |

## 使用例

```yaml
  Conditions:
  - name{n=Ashijin}
```

## エイリアス

- `castername`

# hasowner

**タイプ**: 🟦 エンティティ (Entity)

## 説明

エンティティがオーナーを持っているかどうかをチェックします。

> *This condition checks if the target mob has an owner.*

## 使用例

```yaml
  Conditions:
  - hasowner true
```

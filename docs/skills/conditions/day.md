# day

**タイプ**: 🟩 ロケーション (Location)

## 説明

時刻が昼間（2000〜10000）かどうかをチェックします。

> *Checks if the time in the world at the target location is day, from 2000 to 10000 in-game time.*

## 使用例

```yaml
  Conditions:
  - day true
```

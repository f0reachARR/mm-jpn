# thundering

**タイプ**: 🟩 ロケーション (Location)

## 説明

ワールドで雷が鳴っているかどうかをチェックします。

> *Checks if it's thundering in the target world.*

## 使用例

```yaml
  Conditions:
  - thundering true
```

## エイリアス

- `stormy`
- `isthundering`
- `isstormy`

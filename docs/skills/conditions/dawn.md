# dawn

**タイプ**: 🟩 ロケーション (Location)

## 説明

時刻が夜明けかどうかをチェックします。

> *Checks if the time in the world at the target location is dawn, from 22000 to 2000 in-game time.*

## 使用例

```yaml
  Conditions:
  - dawn true
```

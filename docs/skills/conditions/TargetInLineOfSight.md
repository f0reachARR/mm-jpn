# TargetInLineOfSight

**タイプ**: 🟧 比較 (Compare)

## 説明

ターゲットがキャスターの視線内にいるかどうかをチェックします。

> *Tests if the target entity has line of sight to their target*

## 使用例

```yaml
  Conditions:
  - targetinlineofsight true
```

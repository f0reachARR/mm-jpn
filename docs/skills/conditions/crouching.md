# crouching

**タイプ**: 🟦 エンティティ (Entity)

## 説明

ターゲットエンティティがしゃがんでいるかどうかをチェックします。

> *This condition checks if the target player is crouching or not.*

## 使用例

```yaml
  Conditions:
  - crouching true
```

## エイリアス

- `iscrouching`
- `sneaking`
- `issneaking`

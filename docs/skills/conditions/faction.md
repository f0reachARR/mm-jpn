# faction

**タイプ**: 🟦 エンティティ (Entity)

## 説明

ターゲットエンティティのファクションをテストします。

> *Checks if the target entity is in the specified faction*

## 属性

| 属性 | エイリアス | 説明 | デフォルト |
|------|-----------|------|-----------|
| `faction` | f | The faction to check for |  |

## 使用例

```yaml
TargetConditions:
- faction{faction=MyFaction} true
```

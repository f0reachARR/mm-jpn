# IsMonster

**タイプ**: 🟦 エンティティ (Entity)

## 説明

ターゲットがモンスターかどうかをチェックします。

> *Checks if the evaluated entity is a monster*

## 使用例

```yaml
  TargetConditions:
  - isMonster true
```

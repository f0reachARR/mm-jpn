# スキルメカニクス (Skill Mechanics)

スキルメカニクス（またはベーススキル）はMythicMobsに組み込まれたシンプルなスキルです。モブのSkillリストでこれらの基本スキルを単独で呼び出したり、これらのメカニクスを組み合わせて独自のメタスキルを作成したりすることができます。

メカニクスによっては、エンティティ、場所、またはその両方をターゲットにできます。ターゲットを制御するには[ターゲッター](targeters.md)を使用します。

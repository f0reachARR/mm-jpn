# ターゲッター (Targeters)

ターゲッターはスキルが発動したときに何をターゲットにするかを決定するために使用されます。

ターゲッターは技術的にはオプションですが（デフォルトのターゲッターは通常@triggerになります）、ターゲッターを忘れることはMythicMobsを初めて使う人が犯す最も一般的な間違いの一つです。

ターゲッターがSkillメカニクスで使用される場合、メタスキル内のすべてのスキルは最初のターゲッターを継承します。メタスキル内の個々のメカニクスに独自のターゲッターを付与することで、親ターゲッターを上書きすることができます。

| @Target | @T | キャスターのターゲットをターゲットにする |
| @Trigger | | スキルをトリガーしたエンティティをターゲットにする |
| @NearestPlayer | | 範囲内の最も近いプレイヤーをターゲットにする |
| @WolfOwner | | ウルフのオーナーをターゲットにする |
| @Owner | | モブのオーナーをターゲットにする |
| @Parent | @summoner | モブの親をターゲットにする |
| @Mount | | キャスターのオリジナルマウントをターゲットにする |
| @Father | @dad, @daddy | キャスターモブの父をターゲットにする |
| @Mother | @mom, @mommy | キャスターモブの母をターゲットにする |
| @Passenger | | キャスターモブのライダーをターゲットにする |
| @PlayerByName | @specificplayer | 名前で特定のプレイヤーをターゲットにする |
| @UniqueIdentifier | @UUID | UUIDで特定のエンティティをターゲットにする |
| @Vehicle | | キャスターの乗り物をターゲットにする |

## ターゲッターオプション

ほとんどのターゲッターは括弧内に追加オプションを受け入れます。一般的なオプションには：

- **radius** - ターゲットを検索する半径
- **limit** - ターゲットの最大数
- **sort** - ターゲットのソート方法（nearest, furthest, random, highest_hp等）
- **excludeSource** - trueの場合、キャスターをターゲットから除外
- **faction** - 特定のファクションのエンティティのみをターゲット
- **type** - 特定のエンティティタイプのみをターゲット

```yaml
Skills:
- ignite{ticks=100} @PlayersInRadius{radius=10;limit=3;sort=nearest}
```

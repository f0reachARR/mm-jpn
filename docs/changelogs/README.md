# チェンジログ (Changelogs)

最新のMythicMobsの変更履歴は[公式リポジトリ](https://git.lumine.io/mythiccraft/MythicMobs/-/wikis/changelogs/Changelogs)で確認できます。

## 最近の主な更新

### MythicMobs 5.x

MythicMobs 5.xシリーズでは多くの新機能と改善が行われています：

- 新しいスキルシステムの導入
- パフォーマンスの改善
- 新しいエンティティタイプのサポート
- プレースホルダーシステムの拡張
- 変数システムの拡張
- 新しいメカニクスの追加

### 重要な変更点

チェンジログに記載されている各バージョンには以下の情報が含まれています：

- **新機能**: 新しく追加された機能
- **変更**: 既存の動作の変更
- **バグ修正**: 修正されたバグ
- **削除**: 削除された機能

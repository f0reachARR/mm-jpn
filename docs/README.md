## MythicMobsとは？

MythicMobsは、サーバーオーナーがサーバー上のプレイヤー以外のすべてのエンティティの属性を完全に変更し、独自のカスタム複合システムを作成することを可能にするプラグインです！

このプラグインは、モブのHPと攻撃力を単純に増加させることから、強力かつシンプルなスクリプティングシステムで膨大なアビリティを与えることまで、モブの動作を開発者が完全にコントロールできるようにします。カスタムモブはバニラモブを置き換えたり、様々なスポーン条件を使ってランダムにスポーンしたり、特定の条件やコマンドで起動するカスタムモブスポナーを設定したりすることができます。

究極のボスを作りたいですか？ お任せください。RPG用のシンプルなモブを作りたいだけですか？ MythicMobsがあなたのプラグインです。複雑なMMOからフレンドリーなSMPや高速ミニゲームまで、あなたの想像力だけが限界です。

### MythicMobsでできること

- **モブスキル**:  
  条件、ターゲッター、トリガーを使って、スキルがいつどこで発動するかを正確にコントロールできる強力なYAMLスクリプティングシステムを通じて、モブにカスタムスキルを付与します。

- **モブ属性**:  
  HPや攻撃力、移動速度、戦闘範囲など、すべてのモブ属性を変更します。

- **特殊エフェクト**:  
  モブがカスタムサウンド、パーティクル、特殊エフェクトを使用できるようにします。

- **装備**:  
  任意のモブが装備できる（またはドロップできる）カスタム属性を持つ新しい装備を作成します。

- **ダメージ修飾子**:  
  異なるダメージタイプがモブに与える影響を変更します。例えば、火に免疫を持たせたり、溺死からHPを回復させたり、矢のダメージを軽減させたりできます。

- **モブスポナー**:  
  特定の場所で、オプションのタイマー、条件、属性、クールダウンなどでMythicMobsをスポーンする強力なカスタムモブスポナーを作成します。

- **スポーンコントロール**:  
  指定できる広範な条件リストに基づいて、世界の異なるエリアでカスタムモブやボスがランダムにスポーンするように設定します。バニラのスポーンを完全に置き換えることができます！

- **モブレベル**:  
  スポーンした場所と方法に基づいてモブのレベルをスケーリングし、レベルが上がるにつれてHP、攻撃力、さらには多くのアビリティをスケーリングします！

- **脅威テーブル**:  
  脅威テーブルを使用するようにモブを設定し、ターゲティング動作に対する強化されたコントロールを付与し、RPGの敵に似た動作をモブに可能にします。

- **モブAIコントロール**:  
  モブのAIゴールを変更します。ゾンビにドアを開けさせたり、他のモブファクションのスケルトンと戦うフレンドリーなスケルトンガードを作ったり、羊から逃げるクリーパーを作ったりできます！

- **モブファクション**:  
  モブをファクションの一部にできます！前述のAIコントロールと連携して、他のファクションのモブと戦ったり、プレイヤーを助けたりするモブを作ることができます！

- **ディスガイズ**:  
  LibsDisguisesとの強力な統合により、任意のモブが*何にでも*見えるようにできます！

- **プラグイン互換性**:  
  ModelEngineを含む、多種多様な人気プラグインとの統合。

## ヘルプとサポート

サポートが必要ですか？

- [Discordサーバー](https://www.mythiccraft.io/discord)

MythicMobsは、その範囲の広さから最初は混乱するかもしれない非常に複雑なプラグインです。

最大限に活用するために、Discordサーバーやフォーラムで巨大で活発なMythicMobsコミュニティに参加してください。開発者や無数の他のサーバーオーナーから直接、質問したり、アイデアを得たり、サポートを受けたりすることができます！

## 開発ビルド

開発ビルドはプレミアムユーザーが利用可能で、こちらで見つけることができます：

<https://mythiccraft.io/index.php?pages/official-mythicmobs-dev/>

開発ビルドにはテストされていない機能が含まれており、安定性を意図して作成されていますが、ベータ版として保証はありません。以前のMythicMobs設定を変更または破損する可能性があり、バグや未完成の機能が含まれる場合があります。チェンジログを*完全に*読んでください。
